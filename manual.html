Manual do Usuário

Sistema de Inventário Postos — guia completo para realizar contagens.

Visão Geral

O sistema de Inventário Postos permite realizar contagens de produtos de forma rápida e organizada, salvando os dados diretamente no navegador e gerando arquivos para integração com o sistema de back-office.

index.html — Tela Inicial

Criação e gerenciamento de lotes. Ponto de entrada do sistema.

contagens.html — Contagens

Tela de trabalho: busca produtos, registra quantidades e exporta.

Requisitos

⚠️
Arquivo CSV obrigatório: O sistema depende do arquivo embalagens com categorias.csv para funcionar. Sem ele, não é possível criar lotes nem contar produtos.
O que é necessário
  • Navegador moderno (Chrome, Edge, Firefox, Safari)
  • Arquivo embalagens com categorias.csv disponível no dispositivo
  • Não é necessário internet após o primeiro carregamento
  • Os dados ficam salvos no próprio navegador (IndexedDB)
ℹ️
Os dados são salvos somente neste navegador, neste dispositivo. Limpar os dados do navegador apaga todas as contagens.

Fluxo Completo

O processo de contagem segue estas etapas em ordem:

1
Carregar o CSV na Tela InicialFaça upload do arquivo de produtos para habilitar as categorias.
2

3

4

6

7

✓
Pronto!Volte para criar um novo lote ou continue o existente.

1. Tela Inicial (index.html)

Esta é a primeira tela que você vê. Aqui você carrega o arquivo de produtos e gerencia os lotes existentes.

Carregar o arquivo CSV

ℹ️
O arquivo CSV é carregado automaticamente se já estiver disponível. Caso contrário, aparecerá um botão para selecioná-lo. Após o carregamento, a árvore de categorias fica visível.

Lotes existentes

Os lotes já criados aparecem em forma de cards. Cada card mostra:

  • Nome do lote (usuário + loja + descrição)
  • Dentro do lote: quantidade de produtos selecionados
  • Fora do lote: produtos contados que não estavam na seleção
  • Total contado: número de registros de contagem salvos
Ações nos cards de lotes
Clicar no cardContinua as contagens deste lote (vai para contagens.html)
Ícone de olhoAbre modal mostrando produtos dentro e fora do lote
Ícone de lixeiraExclui o lote e todas as suas contagens (pede confirmação)

2. Criar um Lote

Preencha os campos e selecione as categorias antes de criar o lote.

Campos do formulário

CampoObrigatórioO que preencher
UsuárioSimSelecione seu nome na lista
LojaSimSelecione a loja/filial que está sendo contada
DescriçãoNãoTexto livre para identificar o lote (ex: "Bebidas Corredor A"). Permite ter múltiplos lotes para a mesma loja.

Selecionar categorias

A árvore de categorias é criada a partir do CSV. Clique nos grupos para expandir e marque as categorias desejadas.

Marcar categoria pai Seleciona automaticamente todas as subcategorias abaixo.
Marcação parcial Se apenas alguns filhos estão marcados, o pai fica com estado intermediário.
✅
Após clicar em "Criar Lote", o sistema filtra os produtos das categorias selecionadas e redireciona automaticamente para a tela de contagens.

3. Tela de Contagens (contagens.html)

Esta é a tela principal de trabalho. O nome do lote ativo aparece no cabeçalho.

Antes de começar a contar, preencha onde você está:

CampoO que significaExemplo
TipoSe é contagem de loja ou depósitoLoja / Depósito

4. Registrar Contagens

O processo se repete para cada produto contado:

  1. Digite ou escaneie o código/nome do produto
    Você pode digitar o código de barras numérico ou palavras do nome do produto.
  2. Pressione Enter
    O sistema busca o produto e exibe nome, código e embalagem.
  3. Verifique o produto encontrado e digite a quantidade
    Aceita números decimais (ex: 2.5). Use 0 para descartar sem salvar.
  4. Pressione Enter novamente
    Um som de confirmação indica que a contagem foi salva. Os campos limpam automaticamente.

Busca por nome

Se você digitar palavras em vez de código numérico, o sistema retorna uma lista de produtos correspondentes. Basta clicar no produto desejado.

⚠️
Produto fora do lote: Se o código encontrado no CSV não pertence às categorias do lote, aparece um aviso "Fora do Lote". A contagem ainda pode ser registrada, mas ficará marcada como fora.
🔴
Produto não encontrado: Se o código não existir no CSV, um som de erro toca e a busca não retorna resultado. Verifique se o código está correto.

5. Ver Contagens Registradas

Clique no ícone de prancheta no cabeçalho para abrir o modal com todas as contagens do lote ativo.

Informações exibidas por contagem
  • Descrição completa do produto
  • Código de acesso (CODACESSO)
  • Quantidade por embalagem
  • Localização: Corredor-Coluna-Andar
  • Horário em que foi contado
  • Quantidade contada

Campo de busca no modal

Digite parte do nome ou código do produto para filtrar a lista em tempo real. Por exemplo, digitar "skol" mostra todas as contagens de produtos com "skol" na descrição.

O contador no cabeçalho do modal atualiza para mostrar quantos resultados foram encontrados.

6. Exportar Contagens

Ao clicar em "Exportar Contagens", dois arquivos são baixados automaticamente:

Arquivo CSV

Relatório detalhado. Pode ser aberto no Excel para análise.

Arquivo TXT

Arquivo concatenado para integração com o sistema de back-office.

Campos Explicados

CampoTelaDescrição
UsuárioInicialIdentifica quem está realizando a contagem
LojaInicialFilial/loja sendo contada
Descrição do LoteInicialTexto opcional para diferenciar lotes da mesma loja (ex: "Bebidas", "Higiene")
CategoriasInicialHierarquia de tipos de produto. Define quais produtos pertencem ao lote.

Arquivos Exportados

CSV Detalhado

Nome do arquivo: USUARIO_LOJA_DATA.csv

Contém uma linha por contagem registrada, com todas as informações:

DATA;HORA;USUARIO;TIPO-CONTAGEM;CORREDOR;COLUNA;ANDAR;SEQPRODUTO;CODACESSO;DESCCOMPLETA;QTDEMBALAGEM;QUANTIDADE
2025-01-15;10:35;ALEX;LOJA;A;C;3;12345;7891234567890;CERVEJA BRAHMA 350ML;1;5
2025-01-15;10:36;ALEX;LOJA;A;C;3;12346;7891234567891;REFRIGERANTE COCA 2L;1;10
ℹ️
O arquivo inclui codificação UTF-8 para que o Excel exiba os acentos corretamente. Use ponto e vírgula como delimitador ao importar.

TXT Concatenado

Nome do arquivo: USUARIO_LOJA_DATA_CONCATENADO.txt

Formato fixo para integração com o sistema de back-office. Agrupa e soma contagens do mesmo produto.

Cada linha tem 27 caracteres: 6 zeros + 14 dígitos de código + 7 dígitos de quantidade

0000007891234567890000005 ← COD 7891234567890, Qtd 5
0000007891234567891000010 ← COD 7891234567891, Qtd 10

Dicas e Atalhos

Use o leitor de código de barras Conecte um leitor USB ou Bluetooth. Ele funciona como teclado e pressiona Enter automaticamente.
Ative "Qtde 1" para itens unitários Economize tempo: o Enter no código já salva com quantidade 1.
Localização é salva automaticamente Não precisa preencher corredor/coluna/andar a cada vez. Mude só quando trocar de local.
Busca por palavras Digite parte do nome para encontrar produtos sem código. Ex: "brahma lata" encontra todos que têm essas palavras.
Use tela cheia (F11 ou botão) Evita toques acidentais fora do app em tablets e terminais.
Um produto pode ser contado várias vezes Em locais diferentes ou correções de quantidade. O TXT somará tudo.
Múltiplos lotes na mesma loja Use a Descrição para separar: "Bebidas", "Higiene", "Padaria".
Som de feedback Tom alto = sucesso. Tom baixo = erro. Use fones se o ambiente for barulhento.

Problemas Comuns

Árvore de categorias não aparece

O CSV ainda não foi carregado. Clique no botão de carregar arquivo e selecione o arquivo embalagens com categorias.csv.

Produto não encontrado ao buscar

O código ou nome não existe no CSV carregado. Verifique se o arquivo CSV é o correto e se o código está certo.

Aviso "Produto fora do lote"

O produto existe no CSV mas não pertence às categorias selecionadas no lote. Você ainda pode contá-lo — ele será registrado como "fora do lote" e incluído na exportação normalmente.

Dados sumiram após limpar o navegador

Os dados são salvos no navegador. Limpar "cookies e dados de sites" apaga as contagens. Exporte sempre antes de limpar o navegador.

Botão "Exportar" não aparece

Verifique se existe um lote ativo (nome aparece no cabeçalho). Se o lote não tiver contagens registradas, o botão pode não gerar arquivo.

CSV abre com caracteres estranhos no Excel

Ao abrir o CSV, selecione codificação UTF-8 e delimitador ponto e vírgula (;) no assistente do Excel.
